## Building
1.  Clone this repository: `git clone https://github.com/mtuopensource/Algorithms.git`
2.  Open a terminal and navigate to the root directory of the project
3.  Run `gradlew build`

## Unit Testing
1.  Clone this repository: `git clone https://github.com/mtuopensource/Algorithms.git`
2.  Open a terminal and navigate to the root directory of the project
3.  Run `gradlew test`
